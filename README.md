## SVNovo is a free SVN client written in Lazarus and FPC

**ATTENTION: Very alpha source code!**

SVNovo aims to a simple but complete SVN GUI client .

Developed in [FreePascal](http://freepascal.org/) using [Lazarus](http://www.lazarus-ide.org/).
